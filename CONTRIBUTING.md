# Contributing

Thanks for considering contributing to pyRSS!

## Development setup

- Python 3.9+ recommended
- Install dependency:

  ```sh
  pip install feedparser
  ```

## Running locally

Use the CLI in `src/rsscli.py`:

```sh
python src/rsscli.py --help
```

## Reporting issues

Please include:

- Your Python version
- The command you ran
- The feed URL (if applicable)
- Full traceback or error output

## Submitting changes

1. Fork the repo and create a feature branch.
2. Keep changes focused and scoped to a single topic.
3. Update or add documentation if behavior changes.
4. Run any relevant checks/tests.

## Style

- Prefer clear, descriptive names.
- Keep CLI output concise and actionable.
